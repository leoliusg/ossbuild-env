msys.bat                 This contains some modifications to msys.bat to 
                         support spaces in directories.

msys-x86.bat             Launches msys and modifies its path to use commands 
                         targeting x86 processors.

msys-x86_64.bat          Launches msys and modifies its path to use commands 
                         targeting x86_64 (amd64) processors.

ossbuild.sh              Installed to /etc/profile.d/ to setup a terminal 
                         with ossbuild-specific environment variables, mounts, 
                         etc.
